# Frontend UI Projects

This repository contains small frontend projects built to practice UI layout,
responsive design, and basic JavaScript logic.

## Projects

### Temperature Converter
- Built using HTML, CSS, and JavaScript
- Converts temperature between Celsius, Fahrenheit, and Kelvin
- Implements conditional logic and DOM manipulation
- Styled with custom CSS and gradient background

### Streaming Platform Landing Page
- Responsive landing page built using HTML and CSS
- Uses Flexbox for layout and alignment
- Includes a CSS-only FAQ accordion
- Mobile responsiveness handled using media queries
